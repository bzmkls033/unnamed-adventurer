# 不知名冒险家 🎲

多人联机回合制冒险桌游

## 特点

- 🗺️ 随机生成地图，每局不同
- 🎮 5种小游戏：德州扑克、炸金花、二十一点、幸运轮盘、拍卖会
- 👹 3个Boss挑战
- 💎 3套藏品收集
- 🤝 玩家实时交易
- ⚔️ 道具攻击与防御
- 🏆 多种胜利条件

## 本地运行

```bash
npm install
npm start
```

打开 http://localhost:3000

## 部署

支持一键部署到 Render。

## 技术栈

- Node.js + Express
- Socket.io 实时通信
- 原生 HTML/CSS/JS 前端
